Exercice 2 : Variables et noms corrects

Facile

Corriger les noms de variables selon les conventions JavaScript (camelCase).

Variables corrigées 
// Avant
let user_name = "John";
let UserAge = 25;
let is_active = true;

// Après (camelCase)
let userName = "John";
let userAge = 25;
let isActive = true;

Règle :

En JavaScript, on utilise le camelCase pour les variables : premier mot en minuscule, mots suivants avec majuscule initiale.

Exercice 3 : Incrémentation préfixe/postfixe

Moyen

Comprendre la différence entre ++a (préfixe) et a++ (postfixe).

Code d'exemple 
let a = 1, b = 1;
let c = ++a; // a est incrémenté avant, a=2, c=2
let d = b++; // b est incrémenté après, d=1, b=2

console.log("a:", a); // 2
console.log("b:", b); // 2
console.log("c:", c); // 2
console.log("d:", d); // 1

Résultat :

  • ++a : Incrémente d'abord, puis retourne la valeur
  • a++ : Retourne d'abord la valeur, puis incrémente

Exercice 4 : Le nom officiel de JavaScript

Facile

Demander à l'utilisateur le nom officiel de JavaScript et vérifier la réponse.

Solution avec if...else 
let reponse = prompt("Quel est le nom officiel de JavaScript?");
if (reponse === "ECMAScript") {
    alert("Bonne réponse !");
} else {
    alert("Ne sais pas ? ECMAScript !");
}

Version avec opérateur ternaire :

let result = prompt("Quel est le nom officiel de JavaScript?") === "ECMAScript" 
    ? "Bonne réponse !" 
    : "Ne sais pas ? ECMAScript !";
alert(result);

Exercice 8 : Colorer les diagonales d'un tableau

Moyen

Colorer en rouge toutes les cellules de la diagonale principale d'un tableau.

Solution JavaScript 
let table = document.querySelector('table');
for (let i = 0; i < table.rows.length; i++) {
    let td = table.rows[i].cells[i];
    if (td) {
        td.style.backgroundColor = 'red';
    }
}

Explication :

On parcourt chaque ligne (rows) et on prend la cellule à l'index i (cells[i]), qui correspond à la diagonale.

Partie 4 - Événements et Interactions

Exercice 10 : Horloge colorée avec setInterval

Moyen

Créer une horloge numérique avec boutons Start/Stop.

Fonctionnalité complète 
let timer = null;

function updateClock() {
    const now = new Date();
    document.getElementById('hours').textContent = 
        String(now.getHours()).padStart(2, '0');
    document.getElementById('minutes').textContent = 
        String(now.getMinutes()).padStart(2, '0');
    document.getElementById('seconds').textContent = 
        String(now.getSeconds()).padStart(2, '0');
}

function startClock() {
    if (!timer) {
        updateClock();
        timer = setInterval(updateClock, 1000);
    }
}

function stopClock() {
    if (timer) {
        clearInterval(timer);
        timer = null;
    }
}

// Écouteurs d'événements
document.getElementById('startBtn').onclick = startClock;
document.getElementById('stopBtn').onclick = stopClock;

Exercice 11 : Déplacer un ballon avec la souris

Difficile

Cliquer sur un terrain pour déplacer un ballon à cet endroit.

Code principal 
let field = document.getElementById('field');
let ball = document.getElementById('ball');

field.addEventListener('click', function(event) {
    // Coordonnées relatives au champ
    let x = event.clientX - field.getBoundingClientRect().left 
            - field.clientLeft - ball.offsetWidth / 2;
    let y = event.clientY - field.getBoundingClientRect().top 
            - field.clientTop - ball.offsetHeight / 2;
    
    // Limites pour ne pas sortir du champ
    x = Math.max(0, Math.min(x, field.clientWidth - ball.offsetWidth));
    y = Math.max(0, Math.min(y, field.clientHeight - ball.offsetHeight));
    
    // Déplacement
    ball.style.left = x + 'px';
    ball.style.top = y + 'px';
});

Partie 5 - Fonctions et Méthodes

Exercice 12 : Fonction pow(x, n)

Moyen

Créer une fonction qui élève x à la puissance n.

Différentes implémentations 
// Version avec ** (ES7)
function pow(x, n) {
    return x ** n;
}

// Version avec boucle
function powLoop(x, n) {
    let result = 1;
    for (let i = 0; i < n; i++) {
        result *= x;
    }
    return result;
}

// Version récursive
function powRecursive(x, n) {
    if (n === 1) return x;
    return x * powRecursive(x, n - 1);
}

Exercice 13 : Fonction checkAge (ternaire vs ||)

Facile
Version originale 
function checkAge(age) {
    if (age > 18) {
        return true;
    } else {
        return confirm('Did parents allow you?');
    }
}

Version avec opérateur ternaire :

function checkAge(age) {
    return age > 18 ? true : confirm('Did parents allow you?');
}

Version avec opérateur || :

function checkAge(age) {
    return age > 18 || confirm('Did parents allow you?');
}

Exercice 14 : Créer un calendrier

Difficile
Fonction createCalendar 
function createCalendar(elem, year, month) {
    let date = new Date(year, month - 1, 1);
    let table = '<table><tr>';
    
    // En-têtes des jours
    let days = ['MO', 'TU', 'WE', 'TH', 'FR', 'SA', 'SU'];
    for (let day of days) {
        table += '<th>' + day + '</th>';
    }
    table += '</tr><tr>';
    
    // Cases vides avant le premier jour
    let firstDay = date.getDay();
    if (firstDay === 0) firstDay = 7; // dimanche devient 7
    for (let i = 1; i < firstDay; i++) {
        table += '<td></td>';
    }
    
    // Jours du mois
    while (date.getMonth() === month - 1) {
        table += '<td>' + date.getDate() + '</td>';
        
        // Nouvelle ligne chaque dimanche
        if (date.getDay() === 0) {
            table += '</tr><tr>';
        }
        
        // Jour suivant
        date.setDate(date.getDate() + 1);
    }
    
    table += '</tr></table>';
    elem.innerHTML = table;
}
